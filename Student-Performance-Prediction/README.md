# Data Science Project

## Student Performance Prediction System

### Project Overview
This project predicts student math scores using Machine Learning based on reading and writing scores.

### Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

### Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Linear Regression Model
- Student Score Prediction
- Model Evaluation

### Results
The model predicts student performance with good accuracy using reading and writing scores as input features.
